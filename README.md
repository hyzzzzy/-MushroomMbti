# 버섯 MBTI 테스트
React로 구현한 버섯컨셉으로 한 MBTI 테스트입니다.
## 실행
```bash
npm i
npm start
```
## 환경변수
```
REACT_APP_PUBLIC_URL=https://mushroom-mbti.netlify.app
REACT_APP_KAKAO_SDK_URL=https://developers.kakao.com/sdk/js/kakao.js
REACT_APP_KAKAO_API_KEY=Your API Key
```
